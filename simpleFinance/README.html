% indicates the prompt

Commands

help
Prints help to standard output. 

    % help
load
load <filename>
Loads a file into the current session. 

    % load myFinance
create
create <filename>
Creates a file with the specified file name and loads it into the current session. 

    % create newFinanceFile

show
show <option>
Prints information about the option.
Options:

  • Owner: Prints the owner. 
    
    % show owner
  • Amount: Prints the bank balance. 
    
    % show amount
  • Expenses: Prints the expenses list. 
    
    % show expenses
  • Income: Prints the income list. 
    
    % show income
  • Goals: Prints the goal list. 
    
    % show goals
add
add <expense/income> <description> <amount> <date>
Adds an expense or an income to the list.
<description> One-word description of the expense or income
<amount> amount of expense or income
<date> date of expense or income 

    % add expense Groceries 53.44 12/9/2012
    % add income Job 120 12/9/2012
add goal <description> <amountCurrentlySaved> <goalAmount> <endDate>
Adds a goal to the goal list.
<description> One-word description of the goal
<amountCurrentlySaved> Money currently saved for the goal
<goalAmount> Money needed to complete goal
<endDate> End date for the goal 

    % add goal Travel 0 2000 1/1/2013
remove
remove <expense/income/goal> <description>
Removes an expense, income or goal from the list
<description> One-word description of expense, income or goal 

    % remove expense Groceries
    % remove income Job
    % remove goal Travel
modify
modify amount <goal description> <newAmount>
Modifies the amount currently saved of the goal based on the goal description.
<goal description> One-word description of goal
<newAmount> new currently saved amount 

    % modify amount Travel 500
modify date <goal description> <newDate>
<goal description> One-word description of goal
<newDate> new end date of goal 

    % modify date Travel 1/15/13
set
set owner <first> <last>
Sets the owner of the file to the <first> and <last> name. 

    % set owner Shaishav Parekh
set amount <newAmount>
Sets the bank balance to the <newAmount> 

    % set amount 500.75
exit
Saves the changes made during the finance session to a file and exits the program. 

    % exit
